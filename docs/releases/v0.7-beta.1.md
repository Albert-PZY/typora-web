# v0.7-beta.1

Release date: 2026-05-17
Target: main @ v0.7-beta.1
Stability: Beta prerelease, not production-ready

## Summary

This beta release upgrades rich source editing to CodeMirror 6, improves
Typora-style source reveal behavior for diagrams and math, fixes several
Markdown parsing regressions, and refreshes the public demo so completed
features are visible in both English and Chinese.

## Features

- Replaced the CodeMirror 5 runmode-based code-block highlighter with
  embedded CodeMirror 6 editors for fenced code blocks.
- Added a CodeMirror 6 Markdown source editor for raw source mode, including
  fenced-code highlighting through lazy language loading.
- Listed official CodeMirror language package coverage and exposed language
  candidates through the fenced-code language input.
- Added lazy loading for CodeMirror language support so unsupported or unused
  languages do not block initial editor startup.
- Kept successful Mermaid diagrams rendered by default while allowing users to
  click the diagram preview and reveal the source editor above it.
- Added the same click-to-source reveal pattern for KaTeX block math.
- Updated the homepage demo to cover local files, hard breaks, focus mode,
  typewriter mode, shortcuts, bare autolinks, tables, tasks, HTML source
  blocks, KaTeX, Mermaid, and CodeMirror 6 code blocks.
- Added localized English and Chinese demo documents that switch with the
  website language toggle while preserving user edits once the demo is touched.
- Updated the website brand and browser tab icon to use `public/favicon.svg`.

## Fixes

- Fixed bare `http://` and `https://` URLs so they are recognized as autolinks
  without swallowing trailing punctuation.
- Fixed CJK strong-emphasis parsing where `**xx**`-style content could lose
  adjacent characters because ASCII-only delimiter logic misclassified text.
- Preserved page scroll position when switching between rendered mode and source
  mode.
- Replaced direct HTML block rendering with source-text display, so elements
  such as `<details>` no longer become native browser widgets inside the editor.
- Removed stale CodeMirror 5 token CSS from the default theme.

## Documentation

- Added a Chinese overview to the top of `README.md`.
- Added a Chinese contribution guide to the top of `CONTRIBUTING.md`.
- Updated README feature descriptions for CodeMirror 6 and HTML source-text
  display.

## Tests

- Added coverage for CodeMirror 6 language candidates, async language loading,
  embedded fenced-code editors, and source-mode Markdown highlighting.
- Added regression coverage for Mermaid and math click-to-source reveal.
- Added regression coverage for HTML source-text display, bare URL autolinks,
  CJK strong emphasis, scroll preservation, localized demo content, website
  branding, and documentation language order.

- Version tags continue to use the owner-requested beta format
  `vx.x-beta.x`.

## Known Limitations

- This release is still beta and not production-ready.
- Vite still reports chunk-size warnings because Mermaid, KaTeX, and the
  CodeMirror language-data ecosystem produce large lazy-loaded bundles.
- HTML blocks are displayed as source text by default. A sanitized rendered HTML
  preview remains intentionally out of scope for this milestone.
- Mermaid is still the only diagram engine implemented.
