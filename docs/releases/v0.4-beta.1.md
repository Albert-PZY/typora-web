# v0.4-beta.1

Release date: 2026-05-17
Target: `main` at tag `v0.4-beta.1`
Stability: Beta prerelease, not production-ready

## Summary

This beta release consolidates the Typora-parity backlog work and the follow-up
repository cleanup requested for the Albert-PZY fork. It keeps the editor
native and framework-free while expanding Markdown rendering, editor modes,
custom Typora theme support, local Markdown file workflows, and repository
process discipline.

## Features

- Added inline and block math rendering through KaTeX.
- Added Mermaid fenced-code rendering for diagram blocks.
- Added focus mode and typewriter mode editor controls.
- Added common Markdown editing shortcuts.
- Added runtime import and scoped application of common Typora CSS themes.
- Added local `.md` open/save flows with File System Access API support and
  graceful browser fallbacks.

## Fixes

- Fixed `Shift+Enter` hard breaks so text typed after the break stays on the
  next visual line.
- Fixed Mermaid rendering so valid diagrams hide their source code after a
  successful render.
- Preserved Mermaid source visibility when Mermaid reports a syntax or render
  error.
- Added Mermaid render timeout handling so stuck dynamic loads or renders
  become editable error states instead of staying in a loading state forever.
- Prevented Mermaid NodeView mutation feedback loops by ignoring NodeView-owned
  chrome and diagram DOM mutations.

## Documentation

- Replaced `CLAUDE.md` with Codex-style `AGENTS.md`.
- Added strict Conventional Commits policy in
  `docs/git-commit-convention.md`.
- Added beta release and GitHub Release process documentation in
  `docs/release-process.md`.

## Repository

- Removed GitHub Actions workflows from the project.
- Prepared package metadata for beta release with `package.json` version
  `0.4.0-beta.1`.
- Rewrote repository history so unrelated upstream commits are removed from the
  published `main` branch.
- Removed obsolete upstream version tags from the published repository.

## Breaking Changes

- No public editor API breaking changes are introduced by this beta.
- Repository history is rewritten. Existing local clones should reclone the
  repository or reset their local `main` branch to the new remote history.

## Verification

- `pnpm test` passed.
- `pnpm build` passed.
- `pnpm build:lib` passed.
- `git diff --check` passed.
- Browser smoke test passed for Mermaid success and error states:
  successful diagrams hide `<code>`, syntax errors keep source visible.

## Known Limitations

- This is a beta prerelease and is not production-ready.
- Mermaid is the only diagram engine currently supported.
- Inline parser limitations remain for full nested emphasis and some complex
  link edge cases documented in the README coverage table.
