## Durable Updates

- Recorded that Method-B inline normalization must map text offsets through
  real ProseMirror text-node segments when inline atom nodes, such as task
  markers, appear in a textblock.
- Recorded that serializer extRanges are textContent-based and must not be
  advanced by inline atom node sizes.
- Recorded the CodeMirror 6 fenced-code contract: ProseMirror `code_block`
  content remains authoritative, and embedded CodeMirror changes dispatch
  ProseMirror transactions.
- Recorded the rich block preview contract: Mermaid, KaTeX block math, and
  sanitized HTML previews may hide valid source by default, but invalid source
  must remain visible and editable.
