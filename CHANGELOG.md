# Changelog

## 0.8.0 (2026-05-27)

This release establishes the stable SemVer release baseline for Typora-Web.

### Features

- Preserve the current Typora-style Markdown editing surface, including
  ProseMirror-based WYSIWYG editing, CodeMirror 6 source editing, KaTeX math,
  Mermaid diagrams, sanitized HTML block previews, local file workflows, focus
  mode, typewriter mode, and bilingual website chrome.

### Repository

- Replace beta milestone tags with stable `vX.Y.Z` GitHub Releases.
- Adopt Release Please for automatic release pull requests and generated release
  notes.
- Keep npm publishing gated behind stable release tags.
